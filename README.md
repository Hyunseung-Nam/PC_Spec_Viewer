# PC 사양 확인 프로그램

## 목적
- 하드웨어 사양을 빠르게 확인하기 위한 프로그램

## 실행 환경
- Windows 7/ 10 / 11
- 관리자 권한 불필요

## 기술 메모
- Python 3.7 (Win7 호환)
- PyQt5
- WMI 기반 사양 수집
- 방어 로직 다수 포함

## 히스토리
**v1.4**: UI 비율/폰트 안정화 및 초기 로딩 개선
- Windows 7/10/11 환경에서 DPI 배율 차이에 따른 UI 비율/폰트 크기 흔들림 완화
- 실행 직후 로딩 오버레이 표시로 초기 체감 속도 개선
- RAM 브랜드 표기 개선 (WMI/SMBIOS 문자열 + PartNumber 패턴 기반 추론)
- RAM 사양 표기 구조 변경
  - 동일 항목 중복 시 xN 압축 로직 추가
  - None 기반 fallback 처리로 이중 "정보 없음" 제거
  - RAM 미수집 시 케이스에 따라 '모듈 정보 미제공' 또는 '장착되지 않음'으로 표기하도록 변경
  - RAM 용량 표기 포맷 통일 (8.00GB → 8GB)

**v1.3**: 내부 구조 리팩터링 및 사양 수집 성능 최적화
- SOLID 원칙에 따라 collector/formatter 책임 분리 및 인터페이스(Protocol) 도입하여 유지보수성과 확장성 향상
- WMI 연결 재사용으로 시스템 사양 수집 시 불필요한 연결 생성 횟수

**v1.2**: VGA 메모리 표기 기능 추가 및 사양 정보 정리
- DXGI 기반 전용 GPU VRAM(전용 메모리) 정보 표기 추가
- RAM 총 용량 표기 기준 정리

**v1.1**:

**v1.0**:
